Think Technical Art

I make art, but I make it work too.

I use Blender, Marmoset Toolbag, and a variety of other tools to create game-ready assets for projects in Unity or Unreal Engine.
I handle the full asset pipeline, including modeling, texturing, UV unwrapping, animating, and baking for environment props, vehicles, interiors, and weapons.
I implement assets directly into game engines, set up their animations, character controllers, and audio where needed.
